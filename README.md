# react-native-esme-ui-kit

UI Kit for personal use

## Installation

```sh
npm install react-native-esme-ui-kit
```

## Usage

```js
import EsmeUiKit from "react-native-esme-ui-kit";

// ...

const result = await EsmeUiKit.multiply(3, 7);
```
